# Asta plugins

A single installable plugin, discoverable via `.claude-plugin/marketplace.json`:

- **`asta-tools/`** — every skill and hook lives here.

## Installing

```bash
# Claude Code + Codex (whole plugin, native plugin system, auto-updates)
npx plugins add allenai/asta-plugins

# Any agent (skills only)
npx skills add allenai/asta-plugins
```
